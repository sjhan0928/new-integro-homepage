# Integro Homepage Demo 1

기본적인 Next.js 프로젝트 설정

## 시작하기

먼저 의존성을 설치합니다:

```bash
pnpm install
```

개발 서버를 실행합니다:

```bash
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인할 수 있습니다.

## 기술 스택

- Next.js 16.1.6
- React 19.2.4
- TypeScript
- Tailwind CSS v4
- shadcn/ui (New York style)
- next-themes (다크 모드 지원)
